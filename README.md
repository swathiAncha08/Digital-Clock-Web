# **Digital Clock Web App**

## **Overview**
The **Digital Clock Web App** is a real-time clock application developed using **HTML, CSS, and JavaScript**.  
It displays the current time dynamically and updates every second using JavaScript timers.

## **Key Features**
- **Real-time digital clock**
- **12-hour / 24-hour format toggle**
- **Google-style colorful animated heading**
- **Glowing digital clock display**
- **Starry night animated background**
- **Centered responsive user interface**

## **Technologies Used**
- **HTML** – Structure of the webpage  
- **CSS** – Styling, layout, glowing effects, and background design  
- **JavaScript** – Real-time clock logic and toggle functionality

## **Project Objective**
This project demonstrates essential **frontend development concepts** including:
- **DOM Manipulation**
- **Event Handling**
- **Real-time UI updates**
- **Responsive design principles**
